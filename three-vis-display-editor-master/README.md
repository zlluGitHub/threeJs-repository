# 使用

* 鼠标左键选择
* 鼠标右键旋转视角
* 鼠标中键缩放视角
* `ctrl + 鼠标右键` 移动视角
* F1 - F4 多窗口
* `crtl + z`、 `ctrl + y` 撤销、回复
* x, y, z控制器轴显示
* e, r, t缩放、旋转、平移

# 部署

将index.html中的所有`https://shiotsukikaedesari.gitee.io/three-vis-display-editor`替换为自己服务器的该项目资源路径。
